# urospenezic.BusinessIntelligenceApp
This application does entry level business intelligence logic on X amount of customers. Customers' data is standardised via Z-Score standardisation. It uses KNN classification algorithm to determine customers that are most similar to the given customer. Customer properties are RFM analysis results. 
Start the application using Visual Studio Version Selector. Simply add 4 or more customers and click on Classify. You will be given a standardised version of their respective properties. RFM stands for recency, frequency and monetary. Next to this matrix, you will see a list of customers. Those are the customers most similar to the one selected by the button above. 
This is a wpf application that uses MVVM, repository and observer patern. It is built solely with C# and XAML.
